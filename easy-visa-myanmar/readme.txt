A modern blue, white and black WordPress theme for Myanmar visa guides, flight ticket booking, hotel rental, Letter Service, TM30 Service, travel services and knowledge sharing.

== Installation ==
1. Upload easy-visa-myanmar.zip from WordPress Admin > Appearance > Themes > Add New > Upload Theme.
2. Activate the theme.
3. Go to Appearance > Customize > Easy Visa Home Settings.
4. Set hero text, booking links, and footer text.
5. Create menu from Appearance > Menus and assign to Primary Menu.
6. Optional: Add Visa Guides, Destinations, FAQs, reviews, hero slides, and popup messages from their admin menus.

== Features ==
- Modern homepage with hero section.
- Flight Ticket / Hotel Rent / Easy Pass / Easy Extension / Letter Service / TM30 Service tab system.
- Visa Guides / Blog section on homepage.
- Myanmar Unicode font support using Noto Sans Myanmar plus system font fallbacks.
- Auto SEO meta tags when no SEO plugin is active.
- Search engine readiness: robots meta, sitemap reference in robots.txt, schema JSON-LD and verification meta settings.
- Duplicate SEO prevention for Yoast SEO, Rank Math, AIOSEO and SEOPress.
- Custom post types: Visa Guides and Destinations.
- Guide Categories for Visa Guides: Thailand Visa, TM30, Visa Extension, Letter Service, Travel Tips and Myanmar Language Guides.
- Dedicated /visa-guides/ archive and individual guide pages with Start Request CTA buttons.
- Contact / Inquiry page template with admin-saved inquiries.
- Mobile responsive menu.
- Social media icon tags for Facebook, Instagram and LINE.
- Customizer settings.
- Letter Service form for Myanmar-only recommendation letter requests.
- TM30 Service form with Bangkok, Chiang Mai and Mae Sot regions, contact info, and delivery method selection.

== Search Engine Setup ==
The theme prepares your site for indexing, but search engines must crawl your live public domain.
1. Make sure Settings > Reading > Discourage search engines from indexing this site is unchecked.
2. Submit your sitemap URL: https://yourdomain.com/wp-sitemap.xml
3. Add verification codes in Appearance > Customize > Easy Visa Home Settings.
4. Submit the sitemap in Google Search Console and Bing Webmaster Tools.
